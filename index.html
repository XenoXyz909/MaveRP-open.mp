How to Install

Follow these steps to join GTA:SOUTHSIDE

01

Download Required Clients

Choose one of the following clients to get started:

SA-MP Client
open.mp Client
APC Launcher
Andro Client
02

Install and Configure

Install your chosen client and add our server IP: play.gta-southside.com:7777

Make sure to run the client as administrator for best performance.

03

Register Account

Connect to the server and follow the in-game registration process to create your character.

Complete the tutorial to understand basic server mechanics and rules.
